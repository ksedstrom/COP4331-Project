In order to set up the game run gradlew.bat
In order to set up the server: first install nodejs https://nodejs.org/dist/v16.14.2/node-v16.14.2-x64.msi
then go to the server file and run the following commands:
	npm install
	npm install express --save
	npm install socket.io
	npm install mysql
Then in order to enter online mode, start the server by running server.bat BEFORE running game.bat

You can open the game by running game.bat
